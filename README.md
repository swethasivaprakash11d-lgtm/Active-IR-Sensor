# Active IR Sensor using Arduino UNO

## Overview

This project demonstrates object detection using an Active IR sensor and Arduino UNO. The sensor detects nearby objects by transmitting infrared light and receiving the reflected signal.

## Components Used

* Arduino UNO
* Active IR Sensor Module
* Breadboard
* Jumper Wires
* USB Cable

## Working Principle

The Active IR sensor contains both an IR transmitter and an IR receiver. The transmitter emits infrared light, and when an object is present, the reflected light is detected by the receiver. The Arduino reads the sensor output and displays the object detection status through the Serial Monitor.
